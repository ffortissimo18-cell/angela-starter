# Урок 3. День цикла и энергия по фазам 🌙

**Что добавляем:** бот учитывает день твоего цикла и мягко подсказывает, чего
ждать по энергии — где сегодня может быть подъём, а где лучше беречь себя.

Это опциональный модуль. По умолчанию выключен. Это самый простой урок — по сути
один переключатель.

---

## Шаги

### 1. Таблица уже есть

Таблица `cycle_tracking` создалась ещё в Уроке 1 (она в `schema.sql`). Ничего
делать не нужно.

### 2. Включи модуль

Railway → **Variables** → добавь:

```
ENABLE_CYCLE=true
```

Railway пересоберётся сам.

### 3. Пользуйся

Просто скажи боту в любой момент, например: «сегодня 3 день цикла». Бот:
- запомнит это,
- в утренних чекинах будет учитывать фазу и мягко комментировать энергию.

Фазы, на которые он опирается:
- 1–5 менструация — мягкий режим
- 6–12 фолликулярная — энергия растёт, хорошо начинать новое
- 13–15 овуляция — пик энергии и общения
- 16–20 ранняя лютеиновая — доделывать, наводить порядок
- 21–28+ ПМС — беречь себя, лайт-режим

---

## Хочешь поменять формулировки?

Текст про фазы лежит в `assistant/tools/cycle.py`, переменная `PROMPT_ADDON`.
Можешь переписать своими словами (например, добавить свои наблюдения про
энергию по дням). Редактируется так же, как в Уроке 2.

## Проверь

Скажи боту «сегодня 5 день цикла», потом спроси «какая у меня сейчас фаза?» —
он должен ответить осмысленно.

> Бот никогда не выдумывает день цикла и не настаивает на теме — реагирует
> только когда ты сама называешь день.
