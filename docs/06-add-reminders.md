# Урок 6. Напоминания и недельный обзор ⏰

Два маленьких улучшения, оба включаются одним переключателем.

---

## А. Напоминания

**Что добавляем:** говоришь «напомни через 2 часа выпить воды» или «напомни
завтра в 9 написать Кате» — и бот напомнит в нужный момент.

### Шаги

1. Таблица `reminders` уже создана в Уроке 1 (она в `schema.sql`).
2. Railway → **Variables**:
   ```
   ENABLE_REMINDERS=true
   ```
3. Готово. Бот раз в минуту проверяет, не пора ли напомнить.

### Проверь

Скажи: «напомни через 2 минуты помахать рукой». Через 2 минуты придёт
`⏰ напоминание: помахать рукой`.

---

## Б. Недельный обзор

**Что добавляем:** по воскресеньям бот присылает мягкий обзор недели — тренд
энергии, что удалось, что повторяется, 1–3 фокуса на следующую неделю. Он
строит его из твоих утренних и вечерних рефлексий (поэтому чем регулярнее
чекины, тем полезнее обзор).

### Шаги

Railway → **Variables**:
```
ENABLE_WEEKLY_REVIEW=true
WEEKLY_REVIEW_HOUR=11     # во сколько в воскресенье (по желанию)
```

Готово. Текст обзора можно настроить в `assistant/prompts.py`, блок
`WEEKLY_REVIEW` (как в Уроке 2).
