# Angela Starter — с чего начать 🌿

Это твой личный AI-ассистент, который живёт в Telegram. Утром спрашивает про
фокус и три главные задачи на день, вечером — что удалось. Помнит, о чём вы
говорили. Дальше ты сама достраиваешь его под себя — по урокам, по одному навыку.

## Как это работает (без магии)

```
ты пишешь боту в Telegram
        ↓
   бот → Claude (мозг) → отвечает тебе
        ↓
  всё важное сохраняется в базу (Supabase)
        ↓
  по расписанию утром и вечером бот пишет первым
```

Тебе не нужно уметь программировать, чтобы запустить базовую версию. Нужно
аккуратно пройти шаги и вставить несколько ключей в нужные поля.

## Что понадобится (всё бесплатно или почти)

- **Telegram** — у тебя уже есть
- **Аккаунт Anthropic** (Claude) — мозг бота. Пополняется от $5, расход копеечный
- **Supabase** — база данных, бесплатный тариф
- **Railway** — где бот живёт и работает 24/7, ~$5/мес
- **GitHub** — чтобы скопировать этот проект себе (бесплатно)
